# Travon Calling

A simple static landing page for a lawn care / landscaping business. This project includes a single HTML file, a stylesheet, and a small JavaScript menu helper.

## Files

- `index.html` — main page markup
- `styles.css` — page styling
- `menu.js` — mobile menu / navigation helper
- `assets/` — supporting images and assets

## Local preview

Open the project folder and serve it with a local web server. For example:

```bash
cd /Users/macjeffiii/Documents/GITHUB/travon-calling
python3 -m http.server 8000

# then open:
http://localhost:8000
```

## Notes

- The stylesheet is linked from `index.html`.
- The script is included with `defer` so it loads after the page markup.
- Add or update `styles.css` and `menu.js` to customize the look and behavior.
